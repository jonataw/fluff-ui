# Timepicker

A timepicker integrated into a standard input field. Uses vue2-datepicker under the hood.


Value:

Copy 
<FLTimepicker
  v-model="date"
  label="Time"
  placeholder="Pick a time"
  format="HH:mm" 
  :minute-step="5" 
/>

# Props

Name Type Default Description
value / v-model Required string Value binding.
aria-label string
aria-labelledby string
autofocus boleaan false Focuses the input field automatically on render.
confirm boleaan false Require the user to confirm their time selection.
description string Alias for description-above.
description-above string Text to show above the input field.
description-below string Text to show below the input field.
disabled boleaan false Disables the input field.
error string Error message for this field. Overrides form-inherited errors.
error-messages Record<string, string> Custom error messages mapping for translations etc.
form Form Form object from FLForm.
format Format "HH:mm" The format of the time to be shown in the input field.
hour-options number[] Specifies which hours are selectable.
id string id attribute.
label string Input label text.
minute-options number[] Specifies which minutes are selectable.
name string name attribute.
open boolean | null null Force calendar to be open or closed. If null, the input focus/blur events controls this property automatically.
placeholder string placeholder attribute. If not specified, the label will be used as placeholder.
second-options number[] Specifies which seconds are selectable.
show-hours boleaan true Show the hours column.
show-minutes boleaan true Show the minutes column.
show-seconds boleaan false Show the seconds column.
size "small" | "default" | "large" | string "default" Size of the input field.
time-options TimeOptions Specifies a set range of selectable times.
use12h boleaan false Use 12 hour clock.
value-type "date" | "timestamp" | "format" | Format "date" Data type of the binding value. If the value-type is format, it will inherit the format of the format property.

# Events

Event Arguments Description
input value:? The value the input was changed to. Value will be of the type specified in value-type property.id:string id property of the input. Emitted when the value is changed by the user.

# TimeOptions

The TimeOptions object specifies the following keys: start, end, step and format.

Copy 
{ start: '00:00', step:'00:30', end: '23:30', format: 'HH:mm' }

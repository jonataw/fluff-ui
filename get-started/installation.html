# Installation

To start using Fluff UI in your project, run the following commands in your project directory:

npm i fluff-ui
# yarn add fluff-ui

Import Fluff UI in your Vue main.js file:

import Vue from 'vue';

import FluffUI from 'fluff-ui';
import 'fluff-ui/dist/main.css';

Vue.use(FluffUI);

new Vue({
  render: h => h(App)
}).$mount('app');

# Nuxt

Fluff UI also works with Nuxt. Import Fluff UI by creating a fluff-ui.js/ts file in your plugins folder:

import Vue from 'vue';
import FluffUI from 'fluff-ui';

Vue.use(FluffUI);

Then add fluff-ui to the plugins list in your nuxt.config.js:

export default {
  plugins: [
    '~/plugins/fluff-ui'
  ]
}
